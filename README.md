- ให้นักเรียนติดตั้ง Docker Desktop ในเครื่อง
- เปิด Terminal (หรือ Command Prompt) ในโฟลเดอร์ที่ Clone โค้ดมา แล้วพิมพ์คำสั่ง:
bash


       docker-compose up --build
- รอจนกว่าระบบจะดาวน์โหลดและติดตั้งทุกอย่างเสร็จ (ครั้งแรกอาจใช้เวลาสักพัก)
- เปิดเบราว์เซอร์ไปที่ http://localhost:3001 แล้วเริ่มใช้งานได้ทันที!
